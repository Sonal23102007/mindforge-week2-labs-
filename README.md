Linux / Docker Research Log

1. WSL / Ubuntu

What is Windows Subsystem for Linux (WSL)?

Windows Subsystem for Linux (WSL) is a feature in Windows that allows users to run a Linux environment directly on a Windows system without using a virtual machine. It helps developers use Linux tools, commands, and utilities efficiently.

Why do developers prefer Ubuntu terminal over Windows PowerShell?

Developers prefer Ubuntu terminal because:

- It supports Linux commands widely used in development
- Better compatibility with open-source tools and frameworks
- Easy package installation using apt
- More suitable for server-side and deployment tasks
- Provides a real Linux-like environment

2. Docker Containers

What is a Docker Container?

A Docker Container is a lightweight, portable environment that contains an application along with all its dependencies, libraries, and configurations needed to run it.

Why is it better to run Python inside a Docker Container?

Running Python inside a Docker Container is better because:

- Ensures the same environment across different systems
- Avoids dependency conflicts
- Makes deployment easier and faster
- Provides isolation from the host operating system
- Improves reliability and consistency of applications
